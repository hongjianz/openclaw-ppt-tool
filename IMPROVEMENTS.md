# PPT工具改进记录

## 已完成的改进

### ✅ v1.2.0 (当前版本)

#### 1. 智能分页算法优化 (高优先级)
**状态**: 已完成

**功能**:
- ✅ 按语义单元分页(完整段落、列表),避免内容截断
- ✅ 检测内容溢出并自动创建新页
- ✅ 支持"续页"标记(如"第 1 页/共 3 页")
- ✅ 估算文本行数,合理分配页面空间
- ✅ 保持相关内容在同一页

**使用示例**:
```bash
# 启用智能分页
python main.py -i long_content.md -o output.pptx --smart-paginate

# 自定义每页最大行数
python main.py -i content.md -o output.pptx --smart-paginate --max-lines 10
```

**文件变更**:
- `src/smart_pagination.py`: 新增智能分页模块
- `main.py`: 添加--smart-paginate和--max-lines参数
- `examples/long_content_example.md`: 长文稿测试示例

**算法说明**:
1. 提取内容块(段落、列表、表格、图片)
2. 估算每个内容块的行数
3. 按语义单元分配到页面,避免截断
4. 检测溢出时自动创建新页
5. 添加页码标记(第 X/Y 页)

#### 2. Markdown表格渲染支持 (高优先级)
**状态**: 已完成

**功能**:
- ✅ 识别Markdown表格语法 `| 列1 | 列2 |`
- ✅ 支持表头和数据行
- ✅ 支持列对齐方式 (左/中/右)
- ✅ 转换为PPT原生表格对象
- ✅ 表格样式定制(背景色、字体、边框)

**使用示例**:
```markdown
| 季度 | 销售额 | 增长率 |
|:----:|:------:|:------:|
| Q1   | 1250   | 15%    |
| Q2   | 1480   | 22%    |
```

**文件变更**:
- `src/content_parser.py`: 添加TableData和TableCell类,解析表格语法
- `src/ppt_generator.py`: 添加_add_table()方法渲染表格
- `examples/table_example.md`: 新增表格示例文稿

#### 2. Python 3.6兼容性修复 (中优先级)
**状态**: 已完成

**问题**: `FutureWarning: split() requires a non-empty pattern match`

**修复**:
- 将 `re.split(r'(?<=[。！？.!?])\s*', text)` 改为 `re.split(r'(?<=[。！？.!?])\s+', text)`
- 添加空文本检查避免无效正则匹配

**文件变更**:
- `src/ppt_generator.py`: _split_text_into_paragraphs()方法
